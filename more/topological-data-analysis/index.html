About the thesis

Topological Data Analysis (TDA) combines topology, the mathematical study of shape, with data analysis. The thesis introduces the foundational tools of TDA, including simplicial complexes, homology, and persistent homology, and discusses how these tools are used to study the shape of data in fields like medical imaging and autonomous navigation.
